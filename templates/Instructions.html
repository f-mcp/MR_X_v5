About this Website

This game is a hunt similar to that presented by a treasure map.
Here, though, the hunt takes place entirely virtually, on your device, through the internet.

In order to complete it, you'll have to use various online tools and sites. All of these are free to access, with one definite exception. Without giving too much away, this shouldn't be an issue in most cases.

I should also say, Google Maps (and Street View by extension) is the bread and butter of this game - if you're stuck, these are a good place to start. So also, don't use Apple maps. It sucks.

Below is a more digestible list of pointers for the game's format:

Clues are numbered, but only so that they can refer to each other easily, you can complete them in any order you like. Be mindful of where you leave each clue, as trails carry on from after answers are inputted. For example, if 'Harry Potter' is an answer to a clue, expect 'Harry Potter' to be used later in the game, if not immediately continued from.
If you get an answer right, you'll be awarded a piece of information which will be vital to another point in the game - not necessarily the subsequent clue.
Expect everything to be 'used' once (ie. information given in correct answers will likely only be used once) with the exception of the key presented prior to the clues.
Lastly, take the names given by Google Street View as belonging to those in the photo, even if this is pretty obviously not the case (it's just a bloody game).


Feel free to zap me a message if you get stuck and don't want to give up on a clue just yet.

Other than that, enjoy.

By Fin McPartlin
